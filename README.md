# 創業計画書作成アシスタント

日本政策金融公庫の創業計画書を参考に、1項目ずつ問答しながら下書きを作る静的Webアプリです。

## 使い方

`index.html` をブラウザで開くと動作します。

GitHub Pagesで公開する場合は、このフォルダの中身をGitHubリポジトリにアップロードし、Pagesの公開元を `main` ブランチの `/root` に設定してください。

## 主な機能

- 7項目の問答形式ヒアリング
- 創業計画書の下書き作成
- 厳しめ採点
- 称号表示
- 追加質問によるブラッシュアップ
- 印刷 / PDF保存

## 注意

このアプリで作成される内容は創業計画書の下書きです。正式提出前には、金額根拠、見積書、借入状況、売上計画などを必ず確認してください。
